## 6. 학습 목차

1. 타입스크립트 시작하기
2. 기본 문법 익히기
3. lib.es5.d.ts 분석하기
4. 타입스크립트 프로젝트 설정하기
5. jQuery 타입 분석하기
6. Axios 타입 분석하기
7. React 타입 분석하기
8. Nodejs 타입 분석하기
9. 패키지 직접 타이핑하기

## 7. 스터디 진행 방식

- Express 타입 분석 챕터 제외하고 전체 페이지를 8주안에 완료하는걸 목표로 정해진 분량을 공부
- 매주 같은 Chapter를 공부하고 한명이 발표를 맡아서 진행하고 다른 스터디원들과 의견 공유
